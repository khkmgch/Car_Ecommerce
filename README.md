# Car_Ecommerce

## Description
車の情報を見ることができるEC サイトです。

車のメーカーごとに並べ替えたり、価格順や新着順に並べ替えたりできます。

Vueコンポーネントの練習のため、

各カードの横幅を動的に変更するインプットのコンポーネントを作成してみました。
